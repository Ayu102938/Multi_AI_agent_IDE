# Modern Digital Clock

## Overview
A modern digital clock application built with Python's Tkinter framework.

## Features
- Black background with neon green text display
- Time updates every second
- Modern and stylish design
- Mouse drag window movement
- Semi-transparent window
- Always on top display

## How to Run
```bash
python digital_clock.py
```

## Usage Instructions
- Window dragging: Hold left mouse button and move
- Exit: Close the window

## System Requirements
- Python 3.6 or later
- OS: Windows, macOS, Linux

## Customization
You can modify settings in the `config.py` file:
- Window size
- Color settings
- Font settings
- Update interval
